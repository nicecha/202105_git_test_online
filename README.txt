프로그램 - Git Bash 실행
열린창에서 git 이라고 치면 명령어들 나옴 == 잘 설치됨

git 정보(이메일/ID) 추가
	git config --global user.email "nicecha16@gmail.com"
	git config --global user.name "nicecha"

커밋에 추가할 파일 선택
	git add README.txt

커밋에 상세설명 기재하면 첫 커밋 만들어짐
	git commit -m "설명 추가"

파일 변경 후 두번째 커밋에 상세설명 기재
	git add README.txt
	git commit -m "두번째 설명 추가"

커밋 로그 확인
	git log

다른 커밋으로 시간 여행
	git checkout 581dd2bs 	#7자리이상 복사붙이기

최신 커밋으로 복귀
	git checkout -
